First Steps:

Pulling the first images: ⚠

Please enter on commadline: 

docker pull hello-world
 
docker pull hello-world:linux

List the containers:

 
docker image ls

Running the first Hello World Docker container:

 
docker run hello-world:linux

Always remove the containers first, therefore list the containers:

 
docker container ls

Remove the Hello Word Docker containers by there container names:

 
docker container remove ...

Removing the Hello Word Docker images:

 
docker image rm hello-world:latest
 
docker image rm hello-world:linux

Testing your Docker Desktop installation:

Please enter on commadline: 

 docker image pull nginx:1.29-bookworm 
 docker image pull nginx:1.29-alpine 
 docker container run -d --name nginx-test1 -p 8080:80 nginx:1.29-bookworm 
 docker container run -d --name nginx-test2 -p 8081:80 nginx:1.29-alpine 
 docker container ps

Please enter on commadline of your host system: 

 curl -I localhost:8080 
 curl -I localhost:8081

In case you can run a browser on your host system, please test in your web browser:

http://localhost:8080/
http://localhost:8081/

Clean up afterwards: 

 docker container stop nginx-test1 
 docker container stop nginx-test2 
 docker container rm nginx-test1 
 docker container rm nginx-test2 
 docker image rm nginx:1.29-bookworm 
 docker image rm nginx:1.29-alpine

Hello World Tratition

Please enter on commadline: 

docker pull hello-world:linux

Checking the image status: 

docker images
 
docker image ls
 
docker run hello-world:linux

Checking the container status: 

docker container ls
 
docker container ls -a
 
docker ps
 
docker ps -a

Give the container a name: 

docker run --name hello1 hello-world:linux

Remove the container right after execution: 

docker run --rm hello-world:linux

Get an overview again: 

docker container ls
 
docker container ls -a
 
docker ps
 
docker ps -a

Let's clean up a litte bit: 

docker container remove hello1

Linux on the Rocks

Pulling the ubuntu image. 🐧

Please enter on commadline: 

docker pull ubuntu:24.04

Checking the image status: 🐧 

docker image ls

Starting a new ubuntu container: 🚀 

docker run -it --name ubuntu-test ubuntu:24.04 bash

Inside the ubuntu container: 😉

 
cat /etc/os-release
 
apt update
 
apt install -y curl
 
curl --version

Exiting the ubuntu container: 😇 

exit

Checking the container status: 🐧

 
docker container ls -a

Restarting the ubuntu container: 🚀 

docker container start -ai ubuntu-test
 
exit

Pulling the first three postgres images. 🐘

Please enter on commadline: 

docker pull postgres:17-trixie
 
docker pull postgres:17-bookworm
 
docker pull postgres:17-alpine

Creating the first three postgres containers. 🐘 

docker create --name pstrixie postgres:17-trixie
 
docker create --name psbookworm postgres:17-bookworm
 
docker create --name psalpine postgres:17-alpine

Starting the three postgres containers: 🐘 

docker start pstrixie 
docker start psbookworm 
docker start psalpine

Stopping the three postgres containers: 🐘 

docker stop pstrixie 
docker stop psbookworm 
docker stop psalpine
