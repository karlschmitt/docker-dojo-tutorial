First Steps:

Pulling the first immage:

 
docker pull hello-world
 
docker pull hello-world:nanoserver-ltsc2025

Running the first container:

 
docker image ls

Testing your Docker Desktop installation:

Please enter on commadline: 

 docker image pull nginx 
 docker container run -d --name nginx-test -p 8080:80 nginx 
 docker container ps

Please test in your web browser:

http://localhost:8080/

Clean up afterwards: 

 docker container stop nginx-test 
 docker container rm nginx-test 
 docker image rm nginx

Pulling the first three postgres images. 🐘

Please enter on commadline: 

docker pull postgres:17-trixie
 
docker pull postgres:17-bookworm
 
docker pull postgres:17-alpine

Creating the first three postgres containers. 🐘 

docker create --name pstrixie postgres:17-trixie
 
docker create --name psbookworm postgres:17-bookworm
 
docker create --name psalpine postgres:17-alpine

Starting the three postgres containers: 🐘 

docker start pstrixie 
docker start psbookworm 
docker start psalpine

Stopping the three postgres containers: 🐘 

docker stop pstrixie 
docker stop psbookworm 
docker stop psalpine
